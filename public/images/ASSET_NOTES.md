## Important note about SVG files

The SVG files are exact visual SVG wrappers. They embed the original raster image inside SVG format so the image appearance is preserved exactly. This is the safest way to keep the design visually identical without redrawing it manually as simplified vector paths.

For the website, use PNG/JPG for photos and SVG/PNG for logo depending on layout needs.
